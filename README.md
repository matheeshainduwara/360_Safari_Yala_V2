## ğŸ”„ Customization Guide

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #2C5F2D;        /* Main green */
    --accent: #F59E0B;         /* Gold accent */
    --secondary: #10B981;      /* Secondary green */
}
```

### Updating Content
- **Text:** Edit directly in `index.html`
- **Images:** Replace images in `Assests` folder (keep same filenames)
- **Prices:** Update in the packages section
- **Contact Info:** Modify in the contact section

### Adding More Gallery Images
1. Add images to `Assests` folder
2. Add new `<div class="gallery-item">` with image in HTML
3. Maintains responsive grid automatically
